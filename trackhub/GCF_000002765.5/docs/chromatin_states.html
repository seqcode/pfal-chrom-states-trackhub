Chromatin States

Chromatin states were generated with ChromHMM (Ernst & Kellis, 2012). Figure 1 below shows the chromatin state names and colors followed by the emissions profiles, relative enrichment over gene features, genomic coverage, gene counts associated with each state, and expression values associated with each state. See paper methods for details on how these were calculated.

Figure 1. Chromatin states in the Plasmodium IDC and associated feature enrichments. A) Hidden Markov Model emission probabilities discovered by ChromHMM. B) Relative enrichment of each state at a selection of genomic features. C) The genomic coverage of each state at each timepoint. R= ring stage (10hpi), T= trophozoites (30hpi), and S= schizont stage (40hpi). D) The number of genes that have each state located at the start of the coding sequence at each timepoint. E) Z-score values for the average gene expression value per chromatin state.
